# 🛍️ Outfit.Lupica

Website katalog outfit sederhana dengan tampilan modern, aesthetic, dan responsive.

🔗 **Live Demo:**
https://fhiiieee.github.io/OutfitLupica.github.io/

---

## ✨ Fitur Utama

* 🔍 **Search Produk**
  Cari produk secara real-time

* 🛒 **Popup Detail Produk**
  Menampilkan:

  * Gambar
  * Nama
  * Harga
  * Deskripsi
  * Tombol Shop

* 📱 **Responsive Design**

  * Desktop ✔️
  * Mobile ✔️

* 🍔 **Navbar Interaktif**

  * Hamburger menu (mobile)
  * Highlight menu saat scroll

* 🎨 **UI Aesthetic**

  * Glass effect
  * Background blur
  * Modern card design
